Ir a temario
Port VLAN Membership
Miembro de VLAN en el Puerto - Esta configuración define a qué VLAN pertenece un puerto específico en el switch, lo que determina cómo se comporta con respecto al tráfico de VLAN.
MAC Address Table
Tabla de Direcciones MAC - La tabla de direcciones MAC es una base de datos que mapea direcciones MAC a puertos en un switch, utilizada para el reenvío de tráfico.
VLAN Tagging
Etiquetado de VLAN - El etiquetado de VLAN es la adición de información de VLAN a los paquetes de datos para permitir el tráfico de varias VLAN a través de un solo puerto.
Duplex Mode
Modo Duplex - El modo dúplex se refiere a la capacidad de un puerto para transmitir y recibir datos simultáneamente en una conexión Ethernet (por ejemplo, half-duplex o full-duplex).
Broadcast Storm
Tormenta de Difusión - Una tormenta de difusión ocurre cuando una gran cantidad de paquetes de difusión se propagan rápidamente en una red, sobrecargando los recursos del switch.
Switch Port Security
Seguridad de Puertos del Switch - Switch Port Security es una característica que permite limitar y controlar el acceso a los puertos del switch para prevenir el acceso no autorizado.
STP Root Bridge
Puente Raíz STP - El Puente Raíz STP es el switch que se elige como el punto central en una red con el protocolo Spanning Tree para evitar bucles.
VLAN Hopping
VLAN Hopping - VLAN Hopping es una técnica utilizada por atacantes para acceder a tráfico de VLAN no autorizado al engañar al switch.
WLAN
Wireless Local Area Network - Red de área local inalámbrica que permite la conexión de dispositivos sin necesidad de cables.
SSID
Service Set Identifier - Identificador único de una red inalámbrica que permite a los dispositivos conectarse a la red.
Router
Dispositivo que permite la conexión de una red local a una red más amplia, como Internet, y que gestiona el tráfico de datos.
Access Point
Punto de acceso - Dispositivo que permite la conexión de dispositivos inalámbricos a una red cableada o inalámbrica.
Encryption
Cifrado - Proceso de convertir la información transmitida a través de una WLAN en un formato ilegible para protegerla de accesos no autorizados.
Bandwidth
Ancho de banda - La cantidad máxima de datos que pueden ser transmitidos a través de una red WLAN en un período de tiempo determinado.
802.11
Especificación de estándar para WLANs definida por el IEEE, que incluye varias variantes como 802.11a, 802.11b, 802.11g, 802.11n, 802.11ac y 802.11ax.
Roaming
Cambio de punto de acceso mientras se mantiene la conexión a una red inalámbrica, permitiendo la movilidad de dispositivos.
MAC Address
Dirección MAC (Media Access Control) - Una dirección única asignada a cada dispositivo de red para identificarlo en una WLAN.
Frequency
Frecuencia - La cantidad de ciclos por segundo a la que opera una WLAN, medida en hertzios (Hz).
Modulation
Modulación - El proceso de modificar una señal inalámbrica para transmitir datos a través de una WLAN.
Firewall
Cortafuegos - Un sistema de seguridad que controla el tráfico de red y protege una WLAN de amenazas externas.
Packet
Paquete - Una unidad de datos que se envía a través de una WLAN, que contiene información y se divide en paquetes más pequeños para su transmisión.
Static MAC Address Entry
Entrada Estática de Dirección MAC - Una entrada estática de dirección MAC es una configuración que asigna manualmente una dirección MAC a un puerto específico en un switch.
